# ao3-wrapped

Using [Zaki's AO3 History Exporter](https://github.com/Zaki-1052/AO3-History-Exporter?tab=readme-ov-file) and HTML/CSS/JS to create an AO3 In Review web app that takes a JSON file of a user's reading history and provides crucial findings, like redirecting to a wellness check if Hurt/No Comfort appears in their top tags.
